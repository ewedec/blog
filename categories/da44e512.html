Linux中find命令用法全汇总

Linux基础 Linux 2018/09/15

第一部分:查找名称查找文件的基本查找命令

  • 1.使用当前目录中的名称查找文件
    在当前工作目录中查找名称为test.c的所有文件。

  • 2.在主目录下查找文件
    查找/home目录下的所有文件，名称为test。

  • 3.使用名称或忽略案例查找文件
    找到名称为test的所有文件，并在/home目录中同时包含大写和小写字母。

  • 4.使用名称按类型查找
    在/目录中查找名称为test的所有目录。

find -type f -name test.php

find -type f -perm 777 -print

#-print 格式化输出在文件的名前面加./

find / -type f ! -perm 777

find / -perm 2644

find / -perm 1551

finde / -perm /u=s

find / -perm /u=r

find / -perm /a=x

find / -type f -perm 0777 -print -exec chmod 644 {} \;

#-exec 将前面产生的结果当成参数一个一个的传给后面的命令执行

#-args 将前面产生的结果当成参数一次性穿个后面的命令执行，不过可以用 -n指定个数

#|前面命令的标准输出，当成后面命令的标准输入

find / -type d -perm 777 -print -exec chmod 755 {} \;

find -type f -name “.txt” -exec rm -f {} \;
find -type f -name “.mp3” -exec rm -f {} \;

find /tmp -type f -empty

find /tmp -type d -empty

find /tmp -type f -name “.*”

find / -user root -name test.c

find ~ -user neil

find /home -group Developer

find ~ -user neil -iname “*.c”

find / -atime 50

find / -mtime +50 -mtime -100

find /cmin -60

find / -mtime -60

find / -amin -60

find /-size 50M

find / -size +50M -size -100M

find / -size 100M -exec rm -rf {} \;

find / -type f -name *.mp3 -size +10M -exec rm {} \;
